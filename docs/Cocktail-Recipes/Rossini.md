# Rossini

## Ingredients & Proportion

- 30 ml 草莓果泥
- 70 ml 普羅塞克起泡葡萄酒

## Garnish

none

## How to make

- 冷凍香檳杯
- 將果泥倒入杯中，加入起泡酒攪拌即可

## Notes

- 草莓果泥以及[[Bellini|貝里泥]]所需的桃子果泥做法都很簡單，將新鮮水果搗碎，加入單糖糖漿後放入攪拌機攪拌，用篩子過濾即可
- 第一次製作果泥按照書中建議使用了二十顆新鮮草莓與 50 ml 單糖糖漿，最後得到約 220 ml 的果泥
